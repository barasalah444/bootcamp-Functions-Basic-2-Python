# bootcamp-Functions-Basic-2-Python

# Basic Python Programming Exercises

This repository contains a collection of fundamental Python functions designed to demonstrate basic programming concepts, including list manipulation, loops, and conditional logic.

---

## Functions Overview

### 1. Countdown (`countdown`)
Accepts an integer as an argument and returns a list containing numbers decreasing from the input value down to zero.

### 2. Print and Return (`print_and_return`)
Accepts a list containing two elements. It prints the first element to the console and returns the second element.

### 3. First Plus Length (`first_plus_length`)
Calculates and returns the sum of the first element's value and the total length of the list.

### 4. Values Greater than Second (`values_greater_than_second`)
Evaluates an input list. If the list contains fewer than two elements, it returns `False`. Otherwise, it generates a new list containing only the elements that are greater than the second element of the original list, prints the count of these elements, and returns the new list.

### 5. This Length, That Value (`length_and_value`)
Accepts two arguments representing size and value. It constructs and returns a list populated with the specified value, repeated according to the requested size.
